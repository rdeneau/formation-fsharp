# F♯ Training

Support for a 3-day training course on F♯

## Commandes

Use `marp` ([link](https://marp.app)) to convert markdown slides to HTML or PDF.

### Markdown to HTML

*watch* mode:

```bash
marp fsharp-training-01-bases.md --watch --theme themes/d-edge.css
```

All files:

```bash
marp fsharp-training-01-bases.md --theme themes/d-edge.css
marp fsharp-training-02-functions.md --theme themes/d-edge.css
marp fsharp-training-03-types.md --theme themes/d-edge.css
marp fsharp-training-04-types-comp.md --theme themes/d-edge.css
marp fsharp-training-05-pattern-matching.md --theme themes/d-edge.css
marp fsharp-training-06-collections.md --theme themes/d-edge.css
marp fsharp-training-07-async.md --theme themes/d-edge.css
marp fsharp-training-08-monad.md --theme themes/d-edge.css
marp fsharp-training-09-module.md --theme themes/d-edge.css
marp fsharp-training-10-oo.md --theme themes/d-edge.css
```
