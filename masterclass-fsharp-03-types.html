Notes : F# ne se limite pas aux fonctions ; le puissant système de types est un autre ingrédient clé. Et comme pour les fonctions, il est essentiel de comprendre le système de types pour être à l'aise dans le langage. En plus des types courants de .NET, F# possède d'autres types qui sont très courants dans les langages fonctionnels mais qui ne sont pas disponibles dans les langages impératifs comme C# ou Java.
